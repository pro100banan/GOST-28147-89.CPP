# GOST-28147-89  (C++)

Программная реализация отечественного стандарта блочного шифрования **ГОСТ 28147-89**, выполненная в рамках курсового проекта по дисциплине «Комплексная защита информации».

## Особенности реализации
- **Язык:** C++ (стандарт C++11 и выше).
- **Архитектура:** Объектно-ориентированная (класс `GostEngine`).
- **Обработка данных:** Потоковая (поблочное чтение файлов), что позволяет шифровать файлы любого объема.
- **Безопасность:** Использование криптостойкого генератора случайных чисел `std::mt19937` для ключей и IV.

## Поддерживаемые режимы
1. **ECB** (Electronic Codebook) — режим простой замены.
2. **CNT** (Counter Mode) — режим гаммирования.
3. **CFB** (Cipher Feedback) — режим гаммирования с обратной связью.
4. **MAC** (Message Authentication Code) — выработка имитовставки (16 раундов).

## Структура проекта
- `GostEngine.h` / `GostEngine.cpp` — ядро алгоритма и логика режимов.
- `main.cpp` — консольный интерфейс пользователя и управление файлами.

## Использование
Программа ожидает наличие следующих директорий в корне исполнения:
- `files_plain/` — для исходных файлов.
- `files_encrypted/` — для зашифрованных результатов.

Результат генерации ключа сохраняется в файл `key.bin`.
